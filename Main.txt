주어진 output이 나오도록 CODE TEMPLATE에 적절한 code를 써라. 
Main class에는 // your code here 부분에만 코드를 추가할 수 있다. 
class Outer에는 적절한 code들을 아무 곳에나 삽입 할 수 있다. 
 
INPUT: none
 
OUTPUT:
value: 30
this.value: 20
Outer.this.value: 10
 
CODE TEMPLATE: 
 
public class Main {
	public static void main(String[] args) {
		// your code here 
	}
}

class Outer {
	int value; 
	class Inner {
		int value; 
		void method1() {
			int value; 
			System.out.println("value: " + value); 
			System.out.println("this.value: " + this.value); 
			System.out.println("Outer.this.value: " + Outer.this.value);
		}
	}
}
